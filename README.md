# E-Commerce-Customer-Behavior-Analytics
This project analyzes customer behavior, sales performance, and operational metrics using a dataset containing 17,049 e-commerce transactions from 5,000 customers

The objective was to identify key business insights related to:

Revenue generation
Customer retention
Product performance
Device usage patterns
Delivery performance
Customer satisfaction

The project combines Power BI for visualization and SQL for business analysis.

Tools & Technologies : 
Power BI
SQL (MySQL)
Excel
DAX

Key Insights: 
Revenue Analysis
    Total revenue exceeded 21.7M TRY
    Istanbul generated the highest revenue (5.65M TRY)
    Top three cities contributed over 50% of total revenue
Product Performance
    Electronics generated 10.48M TRY, nearly half of total sales
    Home & Garden and Sports were the next strongest categories
    Customer Retention
    Returning customers contributed 88% of total revenue
    Repeat customers generated 19.19M TRY
Device Analysis
    Mobile users generated 12.03M TRY
    Mobile accounted for the highest order volume
